# Threat Hunting and Incident Response Case Studies

## Overview

This repository contains a collection of practical threat hunting and incident response case studies developed using Splunk and Sysmon telemetry.

The objective of these investigations is to simulate real-world SOC workflows and demonstrate the process of identifying, analyzing, and correlating security events across Windows systems.

These case studies focus on process execution, PowerShell activity, network connections, persistence mechanisms, authentication events, and timeline reconstruction.
